# api-calculo-isr

- Project setup
```
npm install
```

- Compiles and hot-reloads for development
```
npm run serve
```

- Compiles and minifies for production
```
npm run build
```

- Server start for production
```
npm run start
```

### Ejemplo: Uso variables de entorno
```js

// .env file

NODE_ENV = "production";
HOST = "127.0.0.1";
PORT = 4380;
DB_NAME = "myAwesomeApp";

```
